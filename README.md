# Frontend React E-commerce App

This project is a sample e-commerce web application built with React as a first portfolio project.

## Features
- Fetching product data from FakeStore API
- Displaying products in a card-based layout
- Product search functionality
- Product details page for each item
- Add to cart functionality
- Shopping cart page
- User registration form
- User management (add, edit, delete users)
- Product CRUD operations (create, update, delete)
- Global state management using React Context API
- Client-side routing using React Router

## Technologies Used
- React
- React Router
- Context API
- Axios
- JavaScript
- HTML & CSS

## Notes
- The homepage and shopping cart page are responsive.
- This project is created as a portfolio sample, focusing on React concepts and application logic rather than full UI design.
